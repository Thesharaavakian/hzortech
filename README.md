# HzorTech Business Website

![HzorTech Logo](https://via.placeholder.com/150) <!-- optional, replace with your logo if you have one -->

## Overview

HzorTech is a professional business website built with **Django** and **Bootstrap**, designed to showcase IT services, application development, and business solutions. The project is structured to be **scalable, responsive, and production-ready**, making it suitable for deployment on platforms like **AWS EC2**.

This repository contains the full backend and frontend code for the site, including a **contact form** that stores messages in the database and is ready for future notification integrations (Slack, Telegram, or email).

---

## Features

- **Responsive design** using Bootstrap 5  
- **Main pages:** Home, Services, About, Contact  
- **Contact form:** captures name, email, subject, and message  
- **Django-powered backend:** easily extendable with models and views  
- **Template structure:** clean `base.html` with blocks for easy expansion  
- **Ready for AWS deployment:** EC2, RDS, or other cloud hosting  

---

## Tech Stack

- **Backend:** Python 3 + Django  
- **Frontend:** HTML5, Bootstrap 5  
- **Database:** SQLite (local) / ready for PostgreSQL or AWS RDS  
- **Deployment:** AWS EC2-ready
